## 📱 Overview

ActivBharat is a fitness-focused iOS application developed during my internship at ENIL (Entertainment Network India Limited).

The app is designed to help users:

✅ Follow structured workout plans  
✅ Track daily fitness goals  
✅ Monitor training progress  
✅ Engage with a fitness community  
✅ Share achievements and activities

The project was created to strengthen real-world iOS development skills while implementing scalable UI architecture and reusable component-based design.

## 🏗 Architecture

The project follows **MVVM Architecture**.

```text
View
 │
 ▼
ViewController
 │
 ▼
ViewModel
 │
 ▼
Models
```

### Why MVVM?

- Better Separation of Concerns
- Cleaner Code Structure
- Easier Maintenance
- Improved Scalability
- Reusable Business Logic
